# Journal

Welcome to my public journal. This is where I write about my thoughts, experiences, and projects.

## Structure

- `index.html`: The main viewer for the journal.
- `entries/`: A folder containing all journal entries in Markdown format.
- `styles/`: CSS styles for the journal.

## Getting Started

To add a new entry, simply create a new `.md` file in the `entries/` folder and add a link to it in `index.html`.
